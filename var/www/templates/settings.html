Settings (version 1.98)

Player Name

Changes how your Volumio device is called. This affects Player Name, Airplay Name and UPnP Name. A reboot is required for changes to take effect.

Указано имя плеера в сети. Это влияет на имя в строке баузера, сервера и для UPnP. Для вступления изменений в силу требуется перезагрузка. Крайне не рекомендуется изменять!!

Services management

Enable or disable certain Volumio functionalities


Если вы не пользуетесь Airplay (например толлько BubleUPNP) то для меньшей нагрузки на плеер рекомендуется данную опцию отключить!!

$_system_select[shairport1] $_system_select[shairport0]

Если вы хотите воспроизводить музыку с музыкальногого сервера (например OMV) на V.A.M.P. например через приложение BubleUPNP для смарфона, необходимо данную опцию включить!!

$_system_select[upnpmpdcli1] $_system_select[upnpmpdcli0]

Если вы хотите предоставлять музыку, которая находится на плеере V.A.M.P. другим плеерам (например есть андроид приставка или какой-то плеер, которому нужны музыкальные файлы, что находятся на плеере V.A.M.P. и через протокол UPNP\DLNA), то можно включить две опции ниже. UPNP\DLNA Indexing - делает индексацию файлов UPNP, DLNA Library Server - предоставляет доступ к внешним устройствам. Не рекомендуется, так как сильно нагружает процессор плеера. Настраивайте доступ к музыке V.A.M.P. не через UPNP\DLNA, а через SMB протокол

Sound quality tweaks

These profiles include a set of performance tweaks that act on some system kernel parameters.
You can select the ECO-MODE for reduced power consumption.

Вы можете выбрать ЭКО-РЕЖИМ для снижения энергопотребления (например в очень жаркую погоду, или в слабо охлаждаемом помещении).

Compatibility fixes

For people suffering problems with some receivers and DACs.

$_system_select[cmediafix1] $_system_select[cmediafix0]
For those who have a CM6631 receiver and experiment issues (noise, crackling) between tracks with different sample rates and/or bit depth.
A "dirty" fix that should avoid the problem, do NOT use if everything works normally.
